Command Categories

Basic Commands

These commands are essential for navigating and working with the Linux filesystem:

  • ls - List directory contents
  • cd - Change directory
  • pwd - Print working directory
  • cat - Display file contents
  • man - Display manual pages

File Management

These commands help you manipulate files and manage permissions:

  • chmod - Change file permissions
  • chown - Change file ownership
  • mkdir - Create directories
  • rm - Remove files or directories
  • cp/mv - Copy or move files

System Administration

These commands are used for system monitoring and management:

  • ps - Display process status
  • top - Monitor system processes
  • df - Display disk space usage
  • ifconfig - Configure network interfaces
  • kill - Terminate processes
